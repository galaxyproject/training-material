You can tweak tool panel search by configuring boosts in `galaxy.ini`. ```ini tool_name_boost = 9 tool_section_boost = 3 tool_description_boost = 2 tool_label_boost = 1 tool_stub_boost = 5 tool_help_boost = 0.5 ``` You can also manipulate the number of tools returned for any query by changing the `tool_search_limit`. This will display more or fewer tools for any given query. --- ### Tool Panel search configuration 2/2 ```ini # Enable/ disable Ngram-search for tools.
