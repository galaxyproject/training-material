``` --- # Database Queries ``` month | state | job_count ---------------------+---------+----------- 2016-09-01 00:00:00 | deleted | 11369 2016-09-01 00:00:00 | error | 15375 2016-09-01 00:00:00 | new | 1179 2016-09-01 00:00:00 | ok | 165963 2016-09-01 00:00:00 | paused | 933 2016-09-01 00:00:00 | waiting | 9 2016-10-01 00:00:00 | deleted | 13190 2016-10-01 00:00:00 | error | 12539 2016-10-01 00:00:00 | new | 1183 2016-10-01 00:00:00 | ok | 167547 2016-10-01 00:00:00 | paused | 645 2016-10-01 00:00:00 | queued | 75 2016-10-01 00:00:00 | running | 36 2016-10-01 00:00:00 | waiting | 17 (14 rows) ``` --- # Database Queries "I need a list of current main toolshed users and a number of their repos" ```sql SELECT u.username, COUNT(r.id) AS r_count FROM galaxy_user u JOIN repository r ON u.id = r.user_id WHERE NOT r.deleted GROUP BY u.id ORDER BY r_count DESC LIMIT 12;
