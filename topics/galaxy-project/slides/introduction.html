- Galaxy is a big and complicated project, providing highly valued open science resources. - Complexity offers opportunities for different kinds of engagement. - Finding your way is hard at first, because there is so much activity. - This video introduces a field guide to the project ecosystem. - There are descriptive maps, and stories about how new initiatives began. - No single perspective can cover all the complexities. - For individuals who want to navigate and understand the project. --- ## Origins: The context in which Galaxy was created - In 2005, there was a growing flood of commodity sequencing data. - Disrupting much biology and many budgets. - Experienced bioinformaticians were in short supply in biology laboratories. - Fragile, unreliable analyses were run, under intense pressure to publish expensive data rapidly. - Independent scrutiny, or computational replication of a result was usually infeasible - Scientific trust is built by transparency, scrutiny and replication. - All data rich disciplines faced the same critical shortage of reliable analytic capacity - An open, generic solution would help avoid redundant technical effort for every analysis. ??? - The context in which Galaxy originated, in two thousand and five, helps explain a lot. - Sequencing was expensive, but provided valuable scientific insights. - Technical skills needed to analyse raw data correctly, were in very short supply - Investigators were impatient for results to publish from expensive data. - If an analysis cannot be scrutinised, the results cannot gain scientific trust. - Galaxy was our response to this growing shortfall in reliable analysis capacity. - A scalable solution for transparent analysis in any field. --- ## Solution: An application for accessible transparent analysis. - Galaxy was designed as a solution to the growing need for reliable, shareable analysis. - Integrate sharing and computational reproducibility into a generic analysis framework. - XML "wrapper" turns any Linux command line package into an installable tool - Wrapped packages gain framework benefits, such as workflow interoperability - ToolShed repository automates tool wrapper installation for server administrators. - Many "Flavours" are available, to make a framework instance match the needs of local users. - Adaptable to any kind of open science data where packages are available to wrap. ??? - Galaxy was created to make open science analyses, reproducible and transparent. - Solve all those tough challenges once, in a generic scientific workflow application. - Build those core scientific requirements into every analysis and output. - Accommodate any command line analysis package, making it an accessible and interoperable tool. - Make it easy to wrap and install analysis packages as tools, into the generic framework. - Framework flavour, such as Public Health, or Muon Science, depends on the installed tools. - The analysis needs of any field, can be satisfied, by creating a suitable set of tools. --- ## Free analysis services drive software quality improvement and scientific value - Galaxy source code has been running on real hardware since 2005, as part of the solution. - Providing free, reliable analysis services and storage for anyone, anywhere. - Increasingly heavy use provides realistic, non-stop software stress testing - Users contribute insight and experience to improve functionality - Independent developers contribute improvements and help fix bugs - Reliable software engineering practices are encouraged by heavy use at scale ??? - Free analysis services, using Galaxy source code, have been running continuously, since two thousand and five. - It was highly innovative, and quickly attracted users, collaborators, and research grant support. - Running a free, and heavily used service, provides thorough field testing, for the code. - Tens of thousands of scientists generate bug reports, user feedback, and ideas for improvement. - Community contributors provide bug fixes, ideas, and code improvements. - This maintains robust code, and provides increasing value to users over time. --- ## Success driven by collaborations that increase value to scientists - Reliable code powering free analysis services provides many opportunities for collaboration. - Investigators joining the collaboration, such as the GTN, extend and enhance the whole project. - Open collaborations grow organically, as opportunities arise, rather than by design. - Open shared values make it possible for large collaborations to function efficiently. - Scientists benefit from easy access to reliable complex analyses and training - Hundreds of community participants regularly contribute their skills and ideas. ??? - Galaxy continues to grow rapidly, because scientists value it highly - Encouraging new collaborations, brings new capabilities, and new user communities - Organic, self governed growth, produces increasing complexity and diversity - Shared open source community values help maintain engagement, coherence, and stability - Scientists gain free access to resources that they value highly in their work. - Contributor engagement drives growth and sustainability, in Open projects --- ## The Galaxy project is much more than an open source code repository - Open source projects maintain public code repositories and provide support to end users. - Galaxy does that, but also runs project code 24x7, at scale, providing high availability services. - That would be impossible without resources and effort from a global, open collaboration. - Contributors, investigators and grants come from many different institutions. - No single institution controls Galaxy, so participants govern it for themselves. - Only a global community enterprise could provide these services, at no cost to users. - Skilled participant contributions help the project grow, improve and remain sustainable. ??? - An open source repository on Github can be maintained without direct costs. - Free analysis services, consume substantial and expensive computational infrastructure. - They require highly skilled and dedicated personnel, and a global community, for their reliable operation. - The resources needed are provided by grants, supplemented by support from the community. --- ## What part would you like to play in the project?
